# Bank Analytics Project

## Tools Used
- SQL
- Power BI
- Tableau
- Excel

## Objective
Analyze banking data to understand customer behavior, transaction trends,
and business performance.

## Files Included
- SQL queries for data analysis
- Power BI interactive dashboard
- Tableau dashboard
- Dataset (CSV)
- Final presentation (PDF)

## Key Insights
- Year wise loan amount
- Grade and sub grade wise revolving balance
- Total verified vs not verified customers
- State wise vs last credit pull date wise loan amount
- Home ownership vs last payment date stats
